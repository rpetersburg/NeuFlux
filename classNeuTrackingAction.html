Detailed Description

Header file for defining the actions to take at the beginning and end of a track.

The only capability programmed in at the moment is set parentPDG. This helps to identify a track with it's parent track to find where products come from. This modifies the event so that the top track is always pointed at the correct track.

Note
Post Action is still available to be updated.
